What's in v2.0

Core Features

⚡
Dispatch System New

Gary can now orchestrate multi-step tasks across multiple agents. Route work intelligently to Claude Code for coding tasks, Cowork for research and documents, or handle directly — all from a single instruction.

  • Automatic routing: coding → Claude Code, research → Cowork, answers → Gary direct
  • Four-layer guardrail system: debounce, concurrency lock, cooldown, daily cap
  • Push model dispatch — immediate execution, no polling lag
  • Cost tracking per dispatch job in the api_usage table
🔔
Auto-Notifications New

Real-time alerts delivered where you are. Dispatch results, morning briefs, loop completions, and system events now push to Telegram, email, and macOS desktop notifications with configurable priority levels.

  • Three priority tiers: critical (Telegram + desktop), standard (desktop), ambient (badge)
  • Telegram bot integration — voice, photo, and text message ingestion
  • Native macOS notifications via Tauri's notification plugin
  • Smart deduplication prevents notification storms from job retries
🎛
Dashboard Customization Improved

Your mission control, your layout. Reorganize, resize, and toggle every widget. Preferences persist in SQLite — your layout survives restarts and updates.

  • 10-widget grid with per-widget show/hide and Small / Medium / Full size options
  • DashboardCustomizer slide-in panel — LayoutGrid button in the header
  • Layout config stored in settings table under dashboard.widgets
  • New widgets: Weather, Habit Tracker, Loop Streak, Email Highlights, Priority Tasks
📬
Gmail & Calendar Sync Improved

Full two-way Google integration. Unread badges refresh every 5 minutes, body cache allows offline reading, and meeting prep briefs surface automatically before scheduled events.

  • 6-month Gmail sweep on first connect — full body cache in email_bodies table
  • 7-day calendar sync with real-time read/unread badge updates
  • Meeting prep auto-pilot: AI brief generated 15 minutes before each event
  • Google OAuth tokens stored locally in google_auth SQLite table
⌨️
Keyboard Shortcuts New

Everything within reach of the home row. The command palette centralizes navigation, search, and actions. Capture modals are one chord away from anywhere in the app.

The dashboard now runs on a composable widget architecture. Each widget is independently sized, toggled, and themed — making the dashboard feel genuinely tailored rather than templated.

  • Weather Widget — OpenMeteo API, geolocation, 30-min cache, Fahrenheit

Daily Check-ins New

A 4-phase morning ritual built into the app. Contextual questions adapt to your recent activity, mood tracking feeds weekly digest summaries, and insights surface in your Willie Profile.

  • Auto-triggers 1.2 seconds after app ready — once per day, never again until tomorrow
  • 4 phases: focus, energy, priorities, intentions — takes under 90 seconds
  • Insights written to willie_profile table — informs AI responses all day
  • DB migration v16: daily_checkins table

Getting Started

Quick-Start Guide

1
Connect Google Account

Open Settings → Google and click Connect Google Account. Authorize Calendar and Gmail scopes. Gary will run an initial 6-month Gmail sweep in the background — this takes a few minutes on first run.

2
Configure AI Model Routing

Go to Settings → AI and add your Anthropic API key. Background jobs use Gemini Flash via OpenRouter by default (pennies per day). Add an OpenRouter key in Settings → AI → Model Routing to enable it.

3
Set Up Telegram Bot (optional)

Create a bot via @BotFather on Telegram and paste the token into Settings → Telegram. Send yourself a message to get your chat ID. Gary will now receive voice memos, photos, and text from your phone.

4
Customize Your Dashboard

Click the LayoutGrid icon in the dashboard header to open the widget customizer. Toggle widgets on/off and pick S / M / F sizes. Your layout persists automatically across restarts.

5
Complete Your First Check-in

Gary will auto-prompt your daily check-in 1.2 seconds after startup (once per day). Take 90 seconds to complete all 4 phases. Your answers inform Gary's AI context for the rest of the day.

6
Try the Command Palette

Press ⌘K anywhere in the app. Search for notes, loops, commands, and settings. Use ⌘⇧N to quick-capture a note and ⌘⇧L to capture a loop without leaving your current view.

02 — Dispatch System

Intelligent Task Routing at Scale

Gary routes complex work to the right engine automatically — Claude Code for coding tasks, Cowork for research and analysis. Results come back as structured files you can act on.

  • Claude Code dispatches: $0 cost (Max-covered via CLI)
  • 4-layer guardrail system: debounce → concurrency → cooldown → daily cap
  • Push-model dispatch — no polling, zero latency
  • Results ingested automatically as notes, loops, or files

05 — Morning Brief

Your Day, Assembled Before You Wake Up

Every morning Gary generates a personalized daily digest — synthesizing your calendar, open loops, email, and habits into an actionable briefing cached in your local database.

  • Gemini 2.5 Flash Lite — blazing fast, near-zero cost
  • Cached per-day in SQLite — instant load, no repeated API calls
  • Includes Willie Profile context — people, projects, priorities
  • Loop status, habit streaks, and key events all in one view
